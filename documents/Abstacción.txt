Reservacion (Builder)
- long ID
- String mail
- Estado(enum) estado (¿esta usualmente sería creada como pendiente?)(se tendría que usar el confirmar del RecepcionistaService para confirmar)

- LocalDateTime fecha_ingreso (Me di cuenta que se puede usar LocalTime.NOON porque empezamos a recibir a todos al medio dia)
- LocalDateTime fecha_salida (Lo mismo pero sacamos todos a las 12)
- boolean pagado
